# internalPc

> 这是一个基于vue+element-ui做的后台管理

## Build Setup

> 安装的插件有：

    1、element-ui 框架
        npm install --save element-ui

    2、使用axios发送请求
        npm install --save axios

    3、使用scss作为css的预编译
        npm install node-sass --save-dev
        npm install --save-dev sass-loader

    4、使用vuex作为存储库
        npm install vuex --save
    
    
> 启动项目
    npm run dev


> 打包项目
    npm run build
